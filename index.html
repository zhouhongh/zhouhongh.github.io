Bayesian Graph Convolution LSTM for Skeleton Based Action Recognition

ICCV2019 的文章

利用贝叶斯网络提高模型对同一动作不同姿态的适应性

Spatial temporal graph convolutional networks for skeleton-based action recognition

AAAI 2018

ST-GCN

在文章中，作者提出一种卷积的思路，首先确定一个卷积中心（也就是一个关节点）：

\1. 在 spatial 维度上，取与之相邻的点，作为需要参与卷积的点；

\2. 在 temporal 维度上，取前后帧在相同位置的点，作为需要参与卷积的点。

https://github.com/yysijie/st-gcn

Deep Progressive Reinforcement Learning for Skeleton-based Action Recognition

CVPR 2018

DPRL+GCNN

作者提出了一种取关键帧的方法，类似于视频压缩中的取关键帧。因为在骨架序列中前后帧的信息可能会比较冗余，所以只需要选取序列中比较有代表性的关键帧，就可以进行动作的分类识别。所以在 GCNN 之前，作者加入了一个 FDNe t 用来提取关键帧。作者实验证明，运用了取关键帧的方法，能够增加识别的准确率。

IEEE 2018

GEGCN

在文章中，作者提出除了给图中的点做卷积之外，还可以给图中的边做卷积，这就是所说的 Graph edge convolution, 卷积方式如下图所示

2s-NLGCN

采用的是 spectral-based graph convolution networks，并且也使用了双流的网络结构，一个网络处理点（joint）的信息，另一个网络处理边 (bone) 的信息:

Actional-Structural Graph Convolutional Networks for Skeleton-based Action Recognition

pycharm远程控制台乱码

发表于 2020-02-17 分类于 skills

Pycharm 远程访问服务器终端

点击“Tools”->“Start SSH session…”，选择对应服务器即可。

远程执行服务器代码 控制台输出乱码问题

第一步：设置本地环境变量

  • 修改pycharm.exe.vmoptions文件，添加-Dfile.encoding=UTF-8，将本地环境设置为UTF-8

第二步：设置 File Encoding

  • File–>settings–>File Encoding–> IDE Encodings: utf-8, Project Encodings: utf-8, Default Encoding for properties files: utf-8

第三步： 设置 SSH Terminal 选项

  • File –> settings –> SSH Terminal –> Deployment server: 此处是你所使用的环境， Default encoding: utf-8

参考博客： https://www.cnblogs.com/liyaobin/p/10241198.html
