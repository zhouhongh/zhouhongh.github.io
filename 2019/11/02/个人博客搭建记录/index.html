个人博客搭建记录

发表于 2019-11-02 更新于 2020-02-11 分类于 have fun

前言

无意中在网上看到风格简约优雅的博客，感觉很喜欢。对比自己之前使用的csdn博客，觉得csdn的博客还是在外观上不够好看。由于我写博客的目的是为了记录自己学习之路上的一点一滴的进步，同时一些调试程序的记录我也可以在不同的设备上面看到，也方便我分享。不求成为博客大神，只为记录自己。因此一个中意的外观与方便的排版布局可谓十分重要。发现自己看到的那个博客文末有写道由HEXO驱动，便开始查看安装教程，希望自己也可以搭建这样的一个博客。

我使用hexo+github搭建自己的个人博客，并使用了Next的主题。博客本地电脑编辑，可以上传托管到Github仓库，使用网址 username.github.io直接访问。无网络的情况下也可以本地预览。

下面进入正题，开始Hexo+Github+Next的个人博客搭建教学。

搭建环境准备

hexo环境

hexo是一个开源的Github项目，使用它之前，你的电脑需要有两个前提配置

  • Node.js
  • Git

分别到官网进行下载安装即可，选择安装地址后一路默认即可。网址分别如下：

  • Node.js中文官网
  • Git官网

Github仓库

  1. 如果你没有Github账号的话，需要先注册一个。注册Github

  2. 创建github仓库。这个仓库用于你的博客的托管，并支持直接使用网址访问。注意仓库名必须为username.github.io, 即你的github用户名后面跟上.github.io。这个仓库名同时也是你访问博客的网址。如果你不明白如何创建仓库，请按照如下步骤操作。

  • 在你的github主页中点击头像，选择your repositories.
  • 进入到仓库中后，选择new, 填写仓库名，注意这里的仓库名一定是你的用户名后面加上.github.io。

  • 其他默认，点击创建。这时候你的仓库是空的，后面我们会通过一个hexo插件来上传博客到你的仓库。

    开始配置博客站

使用Next主题

hexo自带的主题其实不太好看，我们可以下载其他的主题来优化我们的博客界面，我使用的一款简洁好看的主题叫做Next.

主题使用教程

Hexo 是高效的静态站点生成框架，她基于 Node.js。 通过 Hexo 你可以轻松地使用 Markdown 编写文章，除了 Markdown 本身的语法之外，还可以使用 Hexo 提供的 标签插件 来快速的插入特定形式的内容。在这篇文章中，假定你已经成功安装了 Hexo，并使用 Hexo 提供的命令创建了一个站点。 你可以访问 Hexo 的文档 了解如何安装 Hexo 在 Hexo 中有两份主要的配置文件，其名称都是 _config.yml。 其中，一份位于站点根目录下，主要包含 Hexo 本身的配置；另一份位于主题目录下（themes/next/)，这份配置由主题作者提供，主要用于配置主题相关的选项。 为了描述方便，在以下说明中，将前者称为站点配置文件， 后者称为主题配置文件。
