torchtext用法示例

发表于 2019-11-12 更新于 2020-02-19 分类于 python

torchtext是文本数据预处理的一个包。

torchtext是为pytorch而设计的，但是也可以与keras、tensorflow等结合使用。

torchtext对数据的处理可以概括为Field，Dataset和迭代器这三部分。

  • Field对象指定要如何处理某个字段.

  • Dataset定义数据源信息.

  • 迭代器返回模型所需要的处理后的数据.迭代器主要分为Iterator, BucketIerator, BPTTIterator三种。

关键帧抽取方法调研

发表于 2019-11-06 更新于 2019-11-12 分类于 调研

引言

关键帧抽取方法在视频重建，视频表示，视频检索等方面有重要的应用。在基于单帧的视频生成任务中，选取合适的关键帧来生成预测视频尤为重要。

在设想中，希望实现检测人的身体姿态转换的帧作为关键帧，例如转身、偏头这样的帧。因为基于单帧的视频预测难以复原看不到的信息，所以给定人物在某一视角下的外表就极为重要。利用这样包含人物各个视角下的外表的关键帧，预计会提升生成视频的真实性。
